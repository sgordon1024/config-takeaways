# 4. Build: the Make to code round-trip

## Path A — stay in Make (structure and function)
1. Point **Make Local** at [your repo].
2. Tweak by prompt, direct controls, and code layers.
3. Publish: push to a branch, or hand to a coding agent (MCP + Code Connect). A human merges.

## Path B — onto the canvas (visual and craft)
1. Rebuild the key screen as Figma components, or import the build.
2. Tweak visually plus Motion mode.
3. Regenerate code via Code Connect + MCP.

**Rule:** tweak the build in Make; tweak the system on the canvas.
